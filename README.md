## How this repo was built / 如何建立此 repo

**English:**

1. Add the skill as a Claude Code plugin:
   ```bash
   claude plugin marketplace add gmliao/narrated-design-walkthrough
   claude plugin install narrated-design-walkthrough
   ```
2. Start a Claude Code session, run `/narrated-design-walkthrough` pointing at a design doc
3. Push the generated output — GitHub Actions builds and deploys to Pages automatically

**中文：**

1. 將 skill 加入 Claude Code：
   ```bash
   claude plugin marketplace add gmliao/narrated-design-walkthrough
   claude plugin install narrated-design-walkthrough
   ```
2. 開啟 Claude Code session，執行 `/narrated-design-walkthrough` 並指向設計文件
3. Push 產出結果，GitHub Actions 自動建置並部署到 Pages
