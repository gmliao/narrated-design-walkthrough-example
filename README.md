# narrated-design-walkthrough-example

Reference repo demonstrating the narrated-design-walkthrough skill. The walkthrough explains the skill's own architecture, generated by the skill itself from `docs/design/`.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue)](https://gmliao.github.io/narrated-design-walkthrough-example/)

## How this repo was built

1. `claude plugin marketplace add gmliao/narrated-design-walkthrough`
2. `claude plugin install narrated-design-walkthrough`
3. Start a Claude Code session, run `/narrated-design-walkthrough`, then push the generated output

## Provenance

Last regenerated using narrated-design-walkthrough @ 2026-05-21

## Local preview

```bash
cd docs/walkthroughs && npm install && npm run dev 2026-05-21-narrated-design-walkthrough-architecture
```
