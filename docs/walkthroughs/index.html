Narrated Design Walkthroughs

Turn engineering design docs into 8-minute audio-first walkthroughs you can listen to on a walk.

This demo uses Mandarin Chinese narration generated via the Web Speech API, so you can hear the architecture explained naturally as you follow along.

Subtitles are written as English-readable stage directions, making each slide self-explanatory even with the sound off.

Open walkthrough →

Web Speech API quality varies — Chrome/Edge recommended for Chinese voice.

How to build your own

claude plugin marketplace add gmliao/narrated-design-walkthrough
claude plugin install narrated-design-walkthrough

Then start a Claude Code session and type /narrated-design-walkthrough to generate a walkthrough from your design docs.
