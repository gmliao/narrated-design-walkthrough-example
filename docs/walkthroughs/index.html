Narrated Design Walkthroughs

Turn engineering design docs — or any technical writing — into 8-minute audio-first walkthroughs you can listen to on a walk.

把工程設計文件 — 或任何技術內容 — 轉成 8 分鐘的音訊優先 walkthrough，戴耳機走路也能聽完。

Mandarin narration via the Web Speech API, English-readable subtitles written as engineer-to-engineer stage directions. Each slide is self-explanatory even with the sound off.

Two persona variants ship today: internal design (engineer briefing the architect on a team decision) and external source (engineer presenting an outside resource with own filter and judgment).

使用 Web Speech API 即時產生普通話旁白，字幕以工程師對話的舞台指示語氣撰寫，不開聲音也能獨立閱讀。

目前提供兩種 persona 變體：內部設計（資深工程師向 architect 簡報團隊決策）與外部資源（資深工程師把外部資料挑出對團隊有用的點，附自己的取捨判斷）。

Web Speech API quality varies — Chrome/Edge recommended for Chinese voice.

不同瀏覽器的 Web Speech API 音質有差異，建議使用 Chrome 或 Edge 以獲得最佳中文語音。

How to build your own

自己來一份

claude plugin marketplace add gmliao/narrated-design-walkthrough
claude plugin install narrated-design-walkthrough

Then start a Claude Code session and run /narrated-design-walkthrough pointing at a design doc (internal) or a URL (external).

安裝完後，在 Claude Code session 執行 /narrated-design-walkthrough，指向設計文件（內部）或一個 URL（外部）即可。
